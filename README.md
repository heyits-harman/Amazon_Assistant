# 🤖 Amazon AI Review Assistant

A Chrome extension that analyzes Amazon product reviews using AI and delivers **personalized insights** based on your demographics, use case, and preferences.

---

## ✨ Features

- 🔍 **Automated Review Scraping** — scrapes up to 30 reviews directly from Amazon product pages using a background tab approach to bypass bot detection
- 🧠 **AI-Powered Analysis** — sends reviews to Gemini API for intelligent summarization
- 👤 **Personalized Insights** — tailors pros, cons, and recommendations based on your age, gender, and use case
- 💾 **Persistent Storage** — saves analysis results and user profiles to MongoDB
- ⚡ **Real-time UI** — floating button with expandable panel injected directly on Amazon product pages

## 🚀 Getting Started

### Prerequisites

- Node.js v18+
- MongoDB running locally
- Google Gemini API key ([get one here](https://aistudio.google.com/app/apikey))
- Google Chrome browser

---

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/amazon-ai-assistant.git
cd amazon-ai-assistant
```

---

### 2. Setup the Backend

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=3000
MONGODB_URI=mongodb://localhost:27017/amazon-ai-assistant
GEMINI_API_KEY=your_gemini_api_key_here
```

Start MongoDB:

```bash
mongod
```

Start the backend server:

```bash
npm run dev
```

You should see:
```
Connected to MongoDB
Server running on port 3000
```

---

### 3. Setup the Extension

```bash
cd extension
npm install
npm run build
```

This compiles TypeScript and outputs to `extension/dist/`.

---

### 4. Load the Extension in Chrome

1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked**
4. Select the `extension/` folder
5. The extension is now active

---

## 🧪 How to Use

1. Go to any **Amazon product page** (URL must contain `/dp/`)
2. A floating **🤖 AI Review** button appears in the bottom right corner
3. Click it to expand the panel
4. Fill in your profile:
   - **Age** — e.g. 25
   - **Gender** — Male / Female / Other
   - **Use Case** — e.g. gym, office, gifting
5. Click **Analyze Reviews**
6. Wait while reviews are scraped and analyzed
7. View your **personalized AI analysis**

## 🏗️ How It Works

### Review Scraping
The extension uses a **background service worker** to open Amazon review pages in hidden tabs. This bypasses Amazon's bot detection since it uses a real browser session with proper headers and cookies. Reviews are paginated using Amazon's `nextPageToken` parameter.

### AI Analysis
Scraped reviews along with the user's profile are sent to the Express backend, which constructs a detailed prompt and calls the Gemini API. The response is streamed back to the extension and displayed in the panel.

### Data Persistence
Analysis results and user profiles are saved to MongoDB for future reference.

## 📌 Known Limitations

- Requires the backend server to be running locally
- Amazon occasionally returns bot-check pages which stops scraping early
- Review count may vary depending on Amazon's response
